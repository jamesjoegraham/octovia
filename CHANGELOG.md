# Octavia Changelog

## Style Improvements

- **Transit theme**: Replaced contrast orange cyclic edges with cool teal (`#5BC0BE`) for a cohesive cool-toned palette
- **Bigger arrows**: Marker size increased from 7×7 → 12×12, stroke-width from 2.5 → 3.5
- **Parallel lane routing**: Shared forward-edge segments now offset by 1 grid cell (10px) and connected with 45° diagonal arrows to prevent overlap
